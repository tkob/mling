# Mling: ML FFI Interface Generator

```
Usage: mling [options] file [clang-options]

Options:
  -h, --help            show this help message and exit
  -p PREFIX, --prefix=PREFIX
                        use PREFIX as the structure name
  -s SML, --sml=SML     sml implementation [mlton|poly]
  --so=SO               shared object to be dlopened (for Poly/ML)
```
